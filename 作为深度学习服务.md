# VGGT服务

需要安装包
```
pip install fastapi uvicorn
```

主要基于下面这个命令行预测，拷贝一份到 predict_one.py 进行修改

```
os.system("python vggt/demo_colmap.py --scene_dir=scene_dir --use_ba --fine_tracking")
```

启动服务

```
cd vggt
uvicorn predict_one:app --host 0.0.0.0 --port 8686
```

TypeScript调用服务，不需要传参数

```
curl -X POST "http://localhost:8686/predict"
```
